# 🚀 Roteiro de PITCH: Match.js
*(Focado em Empreendedorismo, Inovação e Tecnologia)*

Este é um roteiro **curto e direto (estilo Pitch de Startup)**. Como é para uma matéria de empreendedorismo avaliada por um professor de programação, o foco aqui é vender a ideia (o negócio) e provar que a tecnologia sustenta essa ideia.

---

## ⏱️ O Pitch (3 a 5 Minutos)

### 1. O Problema (O "Gancho" - 30 segundos)
"Boa noite. O mercado de TI tem um problema crônico: sobram vagas, mas faltam processos seletivos inteligentes. Plataformas tradicionais são poluídas e frustram os dois lados. De um lado, RHs recebem milhares de currículos genéricos. Do outro, desenvolvedores iniciantes sofrem com vagas de estágio que exigem 3 anos de experiência. Foi para resolver essa ineficiência que criamos a **Match.js**."

### 2. A Solução e Inovação (1 minuto)
"A Match.js não é um mural de vagas. É um **SaaS (Software as a Service)** de nicho exclusivo para tecnologia. 
Nossas inovações principais são:
1. **Algoritmo de Match:** O sistema cruza os dados do perfil do dev com os requisitos da empresa e gera uma 'Nota de Match (%)', poupando horas de triagem do RH.
2. **Selo Estágio de Verdade:** Uma iniciativa para democratizar a entrada de juniores. Vagas avaliadas sem exigências irreais ganham nosso selo de aprovação.
3. **Experiência Dark Luxury:** Um design moderno com Glassmorphism pensado para reter a atenção do público de tecnologia."

### 3. Modelo de Negócios / Monetização (Como ganha dinheiro - 45 seg)
"Nosso modelo de negócios é Freemium. Para o desenvolvedor, a plataforma é 100% gratuita. 
A monetização vem do **B2B (Empresas)**. As empresas pagam assinaturas ou taxas avulsas (Plano PRO) processadas via Stripe para terem acesso a vagas ilimitadas, vagas em destaque no topo da página e acesso ao algoritmo completo de cruzamento de dados."

### 4. A Tecnologia (Para brilhar o olho do professor - 45 seg)
"Para viabilizar esse modelo de negócios de forma rápida e segura, a engenharia foi feita com as tecnologias mais modernas do mercado em 2026:
- **Frontend & Backend unificados:** Next.js 16 com TypeScript.
- **Banco de Dados Relacional:** PostgreSQL na nuvem (Aiven) gerenciado via Prisma ORM, garantindo integridade de dados.
- **Segurança & Pagamentos:** Integração de Webhooks seguros usando a API do Stripe e proteção contra injeção de scripts."

### 5. Encerramento (Chamada para ação - 15 seg)
"A Match.js é a ponte definitiva entre os melhores códigos e as melhores empresas do Brasil. Muito obrigado! Podemos abrir para a demonstração da plataforma."

*(Nesse momento, você abre o `localhost:3000` e mostra rapidamente a Home, o Selo de Estágio e a Nota de Match no painel da empresa).*
